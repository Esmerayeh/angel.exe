# ✧ Angel.exe — A Glitched Heaven

> “this is where my thoughts go when they don’t belong to the world”

Angel.exe is not just a website.  
It’s a personal digital diary, a music shrine, a cinema archive, and a fragmented identity space.

A place where aesthetics, emotion, and code intersect.

---

## ✧ Overview

Angel.exe is a **cyber-gothic, angelcore digital diary platform** designed as a deeply personal internet space.

Inspired by early 2000s web culture (MySpace, SpaceHey, VampireFreaks) and blended with modern UI systems, the project explores:

- identity as fragments  
- music as emotional architecture  
- cinema as alternate lives  
- the internet as a place to exist, not just browse  

---

## ✧ Features

### ✦ Identity Fragments
A poetic “About Me” system that presents personality as layered, shifting states rather than a fixed bio.

### ✦ Music Cathedral
- Spotify integration  
- playlist shrine system  
- each playlist treated as a **self-portrait / relic**  

### ✦ Digital Diary (Log Files)
- personal diary system  
- mood-tagged entries (melancholy, obsession, longing, etc.)  
- designed to feel like fragmented thought logs  

### ✦ Cinema Mania Archive
- curated film collection  
- categorized by emotional impact:
  - comfort films  
  - films that changed brain chemistry  
  - films for longing  
  - films for escape  

### ✦ Hall of Selves
A character-based system representing identity through fictional and archetypal figures:
- Misa Amane  
- Nana Osaki  
- Lain Iwakura  
- Ren Honjo  
- original “selves” (cyber angel, soft self, destructive self, etc.)

### ✦ Anime Reliquary
Aesthetic anime archive presented as a shrine rather than a list.

### ✦ Dream Engine
A surreal, interactive page designed for exploration rather than utility.

---

## ✧ Tech Stack

- Frontend: React + modern UI system (via Lovable)
- Styling: Custom CSS (cyber-gothic + angelcore aesthetic)
- Backend (planned): Auth + content control (Supabase / Firebase)
- Media: Spotify embeds + curated assets

---

## ✧ Design Philosophy

This project intentionally rejects:
- minimalism  
- corporate UI patterns  
- purely functional design  

Instead, it embraces:
- maximalist layering  
- emotional interfaces  
- aesthetic-driven navigation  
- “beautiful chaos”  

The goal is to make the site feel like:
> a personal internet cathedral
